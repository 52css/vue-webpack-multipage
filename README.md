# vue-webpack-multipage

> A Vue.js project

## Build Setup

``` bash
# 安装依赖
npm install

# 本地启动并打开第一个模块
npm run local

# 本地启动并打开第一个模块，可指定模块进行开发
npm run local index about

# 测试环境编译
npm run qa

# 测试环境编译，可指定模块进行开发
npm run qa index about

# 预发布环境编译
npm run preview

# 预发布环境编译，可指定模块进行开发
npm run preview index about

# 线上环境编译
npm run online

# 线上环境编译，可指定模块进行开发
npm run online index about
```
